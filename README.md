# SaaS Landing Page (Next.js)

This project is a Next.js landing page starter optimized for SEO using the App Router.

## Run locally

1. Install dependencies:

```bash
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. Open http://localhost:3000

## SEO features included

- Global `metadata` in `app/layout.tsx`
- `sitemap.xml` via `app/sitemap.ts`
- `robots.txt` via `app/robots.ts`
- Semantic sections and heading hierarchy on the landing page

## Customize before launch

- Replace `https://www.your-saas-domain.com` in:
  - `app/layout.tsx`
  - `app/sitemap.ts`
  - `app/robots.ts`
- Replace `hello@your-saas-domain.com` in `app/page.tsx`
- Update product copy, metrics, and testimonials in `app/page.tsx`
